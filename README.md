# dot files

## Configurations of

- neovim
- zsh
- tmux
- vim

## Prerequisites

Make sure you have installed all of the following tools:

- neovim
- fd (fd-find on Ubuntu)
- ripgrep
- zsh
- tmux
- yarn

## Usage

1. Clone source code

```
$ git clone git@github.com:ptienchuan/dotfiles.git ~/.dotfiles
```

2. Run install script

```
$ sh ~/.dotfiles/install.sh
```
